# Swiggy-Sales-Dashboard-Data-Analytics-Project
Project Overview

The Swiggy Sales Dashboard is a data analytics project built using Power BI to analyze food delivery sales data and generate meaningful business insights. The main objective of this project is to transform raw data into interactive visualizations that help understand sales trends, customer behavior, and regional performance.

This dashboard provides a comprehensive overview of key performance indicators such as total sales, average order value, total orders, and customer ratings. By visualizing these metrics, businesses can quickly evaluate performance and identify opportunities for improvement.

Key Metrics

Total Sales: 53.01M+

Average Rating: 4.34

Average Order Value: ₹268.51

Total Orders: 197K+

Dashboard Features

The dashboard includes several visualizations that make the analysis easy and interactive:

1. Monthly Sales Trend
Displays sales performance across different months, helping identify peak and low-demand periods.

2. Daily Order Trend
Shows how order volume changes throughout the week, which can help businesses optimize operations and staffing.

3. Food Category Analysis (Veg vs Non-Veg)
A donut chart comparing total sales by food type to understand customer food preferences.

4. City-wise Sales Analysis
Highlights the top-performing cities contributing the most to overall revenue.

5. State-wise Map Visualization
A geographical representation showing price distribution and regional sales patterns across states.

6. Interactive Filters (Slicers)
Users can filter the dashboard by:

Month

Restaurant Name

State

These filters allow users to explore the dataset dynamically and gain deeper insights.

Tools & Technologies Used

Power BI – Data visualization and dashboard creation

Data Cleaning & Transformation – Power Query

Data Modeling – Creating relationships between datasets

Key Insights

Sales fluctuate monthly with noticeable peak periods.

Certain cities generate significantly higher revenue compared to others.

Non-veg food items contribute a larger share of sales compared to veg items.

Order volume varies across different days of the week.

Conclusion

This project demonstrates how data visualization can turn raw data into actionable insights. The dashboard helps businesses better understand customer preferences, regional demand, and overall performance, enabling more informed decision-making.
