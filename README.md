# 🍔 Swiggy Sales Dashboard | Data Analytics Project (Power BI)  

## 🚀 Project Overview  
The **Swiggy Sales Dashboard** is an interactive data analytics project built using Power BI to analyze food delivery sales data and uncover key business insights.  

This dashboard transforms raw data into intuitive visualizations, helping businesses understand **sales trends, customer behavior, and regional performance** for better decision-making.  

---

## 🎯 Objectives  
- Analyze food delivery sales performance  
- Track key KPIs such as sales, orders, and ratings  
- Understand customer preferences (Veg vs Non-Veg)  
- Identify top-performing cities and regions  

---

## 📊 Key Metrics  
- 💰 **Total Sales:** 53.01M+  
- ⭐ **Average Rating:** 4.34  
- 💵 **Average Order Value:** ₹268.51  
- 📦 **Total Orders:** 197K+  

---

## 📊 Dashboard Features  

### 📅 Monthly Sales Trend  
- Tracks sales performance across months  
- Helps identify peak and low-demand periods  

### 📈 Daily Order Trend  
- Shows variation in order volume throughout the week  
- Useful for operational and staffing decisions  

### 🥗 Food Category Analysis  
- Comparison between **Veg vs Non-Veg** sales  
- Non-veg items contribute a higher share of revenue  

### 🌆 City-wise Sales Analysis  
- Highlights top-performing cities  
- Identifies major revenue contributors  

### 🌍 State-wise Map Visualization  
- Geographic view of sales distribution  
- Helps identify regional demand patterns  

---

## 🎛️ Interactive Features  
Users can dynamically explore the dashboard using filters:  
- Month  
- Restaurant Name  
- State  

---

## 🛠️ Tools & Technologies  
- **Power BI** – Dashboard development  
- **Power Query** – Data cleaning & transformation  
- **Data Modeling** – Relationship building  
- **Data Visualization** – Insight presentation  

---

## 🚀 Key Insights  
- Sales show **monthly fluctuations with peak demand periods**  
- Certain cities contribute significantly higher revenue  
- **Non-veg items dominate overall sales**  
- Order volume varies across different days of the week
